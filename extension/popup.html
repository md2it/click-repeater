    SHORTCUTS

    To run / stop the extension:

    1. Press: Ctrl+Shift+X
      On Mac: Cmd+Shift+X
    2. Release the keys
    3. Then press M

    The 3-step shortcut is not obvious.

    But it is safer and avoids conflicts with other apps.

    To stop: Esc

    MACROS REPEATER

    • Records clicks on a page
    • Repeats recorded clicks
    • Position mode: clicks recorded coordinates
    • Element mode: clicks page elements
    • Visible mode: shows movement and clicks
    • Stealth mode: works without visualisation
    • Repeats a macros up to 999 times
    • Execution speed: 0.25×, 0.5×, 1×, 2×
    • "Default" macros can be launched with a shortcut
    • Doesn't use the network
